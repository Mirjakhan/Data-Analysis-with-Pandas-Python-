# Data Analysis with Pandas

## Description
This project demonstrates basic data analysis using the Pandas library in Python. It includes loading a dataset, performing basic statistics, and visualizing data.

## Features
- Load and display a dataset
- Perform basic statistical analysis
- Visualize data using matplotlib

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/data-analysis-pandas.git
    cd data-analysis-pandas
    ```
2. Create a virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
    ```
3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## How to Run
1. Place your dataset file (e.g., `data.csv`) in the project directory.
2. Run the script:
    ```bash
    python analyze_data.py
    ```
